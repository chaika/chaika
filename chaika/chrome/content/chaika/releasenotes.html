chaika release notes

chaika が新しいバージョンへと更新されました。

ご利用ありがとうございます。
公式サイト等へのリンクはページ下部をご覧ください。

1.8.0 - 2015/XX/XX

NEWS

  • WebExtensions API の発表に伴い，chaika の開発は無期限停止となります

板一覧 (BBSMENU, サイドバー)

  • 複数の BBSMENU を合成して表示することができるようになりました (#246)

    詳しい設定の方法は BBSMENUの編集 をご参照下さい.

  • それぞれのサイトの Favicon がアイコンとして表示されるようになりました (#250)
  • 板一覧のなかに区切り線を表示することができるようになりました
  • 内蔵の検索プラグインとして, dig2ch.net, refind2ch.net 向けの検索プラグインを追加しました (#251)
  • 1ヶ月おきに自動的に BBSMENU を更新するようになりました
  • 外部板を追加する際に, favorite_boards.xml が Firefox 内蔵のエディタである スクラッチパッド で開くようになりました (#283)

    extensions.chaika.bbsmenu.open_favs_in_scratchpad を false に変更することで, 以前の挙動に戻して好みのエディタで編集することができるようになります.

  • デフォルトの検索エンジンが dig.2ch.net に変更されました (#251)
  • デフォルトの BBSMENU 取得元が http://kita.jikkyo.org/cbm/cbm.cgi/20.p0.m0/-all/bbsmenu.html に変更されました (#303)
  • コンテキストメニューだけで有効な検索プラグインも設定画面に表示されるようになりました
  • search.2ch.net 向けの検索プラグインが削除されました (#251)
  • 一部の掲示板が chaika を用いて開かれない問題を修正しました
  • 文字サイズを中・大にしたときの表示が改善されました

掲示板への対応

  • 多くの外部板を板として認識するようになりました
  • ID のないしたらばの板にて無意味な ID が表示されてしまう問題を修正しました
  • 古いスキンにおいて, 1.7.0 から即時あぼーんが動作しなくなっていた問題を修正しました (#100)

スレッドの表示 (ローカルサーバ)

  • 使用するポートが使えない場合に自動的に他の空いているポートを探すようになりました (#264)

    この機能は設定画面から OFF にすることができます.

  • 使用するポートを設定画面から変更することができるようになりました

    使用するポートをアプリケーションごとに設定する extensions.chaika.server.port.firefox などは廃止されました. ポートを変更して chaika を使用されていた方は再度設定をお願い致します.

  • 使用するポートを起動毎にランダムに選択する機能が追加されました

デフォルトスキン

  • 自分の書き込みを登録し, それに対する返信が投稿されたときに通知を行う機能を追加しました (#295)

    レス番号を右クリックして表示できるメニューから設定を行うことができます. また, スキンの設定画面から登録済みの内容を編集することができます.

  • ハイライトされたレスへ移動するためのショートカットキー(u, i)を追加しました (#298)
  • 逆参照などを処理する際に, 一つのレス内のアンカ数やアンカの範囲を制限することができるようになりました (#223)
  • ID の代わりに発信元やホスト名が表示される掲示板においても発言数カウントが動作するようになりました (#300)
  • 新しい Firefox のデザイン Chameleon Style を参考に, ハイライトされたレスのデザインを変更するとともに, スキン内で使用していた色の数を減らしました (#296)
  • 設定画面のデザインが変更され, OK ボタンが押しやすくなりました
  • 意図しないときにショートカットキーが反応してしまう問題を修正しました (#297)
  • >>1000 を超えるレスに対しポップアップや発言数カウントが動作しない問題を修正しました (#266)

Electrolysis (e10s)

  • Electrolysis (e10s, マルチプロセス化 Firefox) が有効な Firefox 上での動作に対応しました． (#100)
  • e10s の制約から，chaika:// から始まるいくつかの URL が変更されました

    以前の URL へアクセスすると，自動的に新しい URL へリダイレクトされます．

    • 板が chaika://board/[URL] から chrome://chaika/content/board/page.xul?url=[URL] へと変更されました
    • ログマネージャが chaika://log-manager/ から chrome://chaika/content/board/log-manager.xul へと変更されました
    • トラブルシューティング情報が chaika://support/ から chrome://chaika/content/support.xhtml へと変更されました
  • e10s への対応の難しさから, 複数箇所を選択し, コンテキストメニューからそれぞれ別の条件として NGEx に登録する機能が削除されました (#277)

その他

  • コンテキストメニューの項目に現在選択している文字列が表示されるようになりました (#281)
  • スレッド一覧, ログマネージャ, リリースノートにて Firefox の新しいデザインである Chameleon Style を導入しました (#272)
  • スレッドリダイレクタの有効/無効の切り替えに際して再起動が不要になりました
  • いくつかのデフォルト設定値が変更されました (#240)

    変更されたものは以下の通りです. これらは 使用状況のアンケート に基づく変更です.

    • 「スレッド一覧を開いた時に自動的に一覧を更新する」をデフォルトで有効にしました
    • 「2ch スレッド URL を自動的に chaika で開く」をデフォルトで有効にしました
    • 「ウィザードを自動的に閉じる」をデフォルトで有効にしました
    • スレッド一覧において「新しいタブで開く」をデフォルトで有効にしました
    • 「Be アイコンの表示」をデフォルトで有効にしました

  • ソースコードを整理し, 非標準な機能の削除を進めました (#267, #269)
  • minVersion が Firefox 38 に引き上げられました (#85, #275)

    デフォルトのショートカットは以下のとおりです。

    • 選択したスレッドを開く: Enter
    • 選択したスレッドを新規タブで開く: Ctrl+Enter, Alt+Enter
    • スレッド一覧を更新する: r
    • 検索欄にフォーカスを移す: f
    • 下のスレッドへフォーカスを移す: j, ↓
    • 上のスレッドへフォーカスを移す: k, ↑
    • 1ページ分下へ移動する: Space
    • 1ページ分上へ移動する: Shift+Space

    • ウィンドウの高さが低い場合にオプションが見切れてしまう問題を修正しました
